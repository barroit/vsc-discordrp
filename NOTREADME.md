# Discord Rich Presence with Git

Monitors editor state and Git repository status, reflecting them directly in
Discord Rich Presence. Fast, accurate, and lightweight.

## Features

* **Advanced Git Integration:** Accurately reflects repository state. Displays
  active branches, detached HEAD states, unborn branches, and ongoing rebase
  operations.
* **Precise Context Tracking:** Broadcasts active file and current line number
  in real-time.
* **Dynamic Assets:** Automatically switches presence icons based on the current
  repository state or file type in focus.

## Requirements

* Git can be run in the default shell.
* A running instance of the Discord desktop client.
